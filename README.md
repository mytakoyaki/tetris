# 🎮 ClaudeTetris

**PCユーザー専用の革新的テトリスゲーム**

従来のテトリスに「ポイント」と「フィーバーモード」を追加した、運要素を戦略で乗り越える新しいテトリス体験をお楽しみください。

## 🚀 はじめに

ClaudeTetrisは、PC全画面プレイに最適化されたWebベースのテトリスゲームです。独自のポイントシステムにより、運に左右されがちなテトリスを戦略的なゲームに変えました。

### 推奨環境
- **画面サイズ**: 1200px以上のモニター
- **ブラウザ**: Chrome, Firefox, Safari, Edge（最新版）
- **操作**: キーボード必須

## 🎯 ゲームの特徴

### 💎 ポイント＆エクスチェンジシステム
- ブロックを設置するたびに**10ポイント**獲得
- 貯めたポイントで**NEXTブロックを交換**可能
- 運命を自分の手でコントロール！

### 🔥 フィーバーモード
- **20ブロック設置**で自動発動
- **30秒間**の無敵タイム
- **4倍スコア** + **無料交換**

### 🏆 段位システム
- 無段から永世名人まで**14段階**
- リアルタイムで段位昇格を表示
- 高スコアを目指してランクアップ！

## 🎮 操作方法

### 基本操作
| キー | 動作 | 詳細 |
|------|------|------|
| `←` `→` | 左右移動 | ブロックを左右に移動 |
| `↑` | 回転 | ブロックを時計回りに回転 |
| `↓` | ソフトドロップ | ゲージ分だけ早く落下（0.5P/行） |
| `スペース` | ハードドロップ | 即座に底まで落下（1P/行） |
| `C` | ホールド | ブロックを保留（1回のみ） |
| `E` | 交換 | NEXTブロックを変更（30P消費） |

### 特殊操作
| ボタン | 動作 |
|--------|------|
| `⟲` (画面右上) | 縦横レイアウト切り替え |

## 🎲 ゲーム仕様

### スコア計算
| ライン数 | 基本スコア | レベル倍率 | フィーバー時 |
|----------|------------|------------|--------------|
| 1ライン | 100点 | ×レベル倍率 | ×4倍 |
| 2ライン | 300点 | ×レベル倍率 | ×4倍 |
| 3ライン | 500点 | ×レベル倍率 | ×4倍 |
| 4ライン（テトリス） | 800点 | ×レベル倍率 | ×4倍 |

### ポイントシステム
- **ブロック設置**: 10P
- **ソフトドロップ**: 0.5P/行
- **ハードドロップ**: 1P/行
- **交換コスト**: 30P → 45P → 65P → 90P → 120P（累積）

### レベル＆スピード
- **自動レベルアップ**: 30秒ごと
- **最大レベル**: 30
- **落下速度**: レベルに応じて加速

### 段位システム
```
無段 → 初段 → 二段 → 三段 → 四段 → 五段 → 六段 → 七段
→ 八段 → 九段 → 十段 → 名人 → 竜王 → 永世名人
```

## 🏆 上達のコツ

### 初心者向け
1. **まずはポイントを貯める**
   - 無理にライン消去を狙わず、ブロック設置でポイント獲得
   - 30P貯まったら積極的に交換を使う

2. **フィーバーモードを活用**
   - 20ブロック設置でフィーバー発動
   - フィーバー中は無料で何度でも交換可能
   - スコア4倍のチャンスを逃さない

3. **ホールド機能を使う**
   - Cキーでブロックを保留
   - I字ブロックは4ライン消去用に保存

### 中級者向け
1. **T-Spinをマスター**
   - T字ブロックの特殊回転でボーナススコア
   - 通常の3倍のスコアが獲得可能

2. **コンボシステムを理解**
   - 連続でライン消去すると倍率アップ
   - Back-to-Backで1.5倍ボーナス

3. **交換タイミングを最適化**
   - フィーバー直前は交換を控える
   - 累積コストを意識して計画的に使用

### 上級者向け
1. **パーフェクトクリア狙い**
   - 全消しで25,000点の大ボーナス
   - 序盤の低い積み上げから狙う

2. **実績システムを活用**
   - 15種類の実績でポイントボーナス
   - カテゴリ別に効率的に達成

3. **段位昇格を目指す**
   - 各段位の必要スコアを把握
   - 長期戦略でハイスコアを狙う

## 📱 レイアウト切り替え

```
[LEVEL]    [フィールド]  [POINTS]
[FEVER]                 [NEXT]
                        [STATS]
```

### 縦向きレイアウト
```

```

画面右上の`⟲`ボタンでいつでも切り替え可能！

## 🎯 スコアアタックのコツ

### 高スコアへの道筋
1. **序盤**: ポイント蓄積重視
2. **中盤**: フィーバーモード最大活用
3. **終盤**: T-Spin・コンボで大量得点

### フィーバーモード活用法
- 発動前にテトリス準備を整える
- フィーバー中はライン消去に集中
- 交換は積極的に使って理想形を作る

### 段位昇格のタイミング
- 各ゲームで少しずつスコアを伸ばす
- フィーバーモードでの大量得点が鍵
- 実績達成によるボーナスポイントも重要

## 🏅 実績システム

15種類の実績が用意されており、達成するとボーナスポイントが獲得できます：

- **基本実績**: ライン消去、ブロック設置
- **スコア実績**: 高スコア達成
- **テクニカル実績**: T-Spin、コンボ
- **特殊実績**: パーフェクトクリア
- **チャレンジ実績**: 高難度目標
